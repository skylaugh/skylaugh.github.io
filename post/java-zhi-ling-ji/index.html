Java指令集

2019-12-17

举个例子:

public class HelloWorld {
    public static void main(String[] args){
        String a = "Hello";
        String b = "World";
        System.out.println(a+b);
    }
}

编译、查看指令集：

javac HelloWorld.java
javap -c HelloWorld.class
